# Modules

This directory contains the modules for the C2M Dashboard application. Each module is responsible for a specific feature or functionality within the dashboard.

## Modules Overview

### /auth
This module handles authentication-related features, including login, registration, and user management.

### /dev-dashboard
This module provides a development dashboard for engineers, allowing them to manage and monitor development-related tasks, and have some pages for PoCs and related shit. If you need something to test, put it here and to /app/(auth)/(engineering)/dev/....

### /footage-uploader
This module handles the uploading and management of video footage.

Right now there is only uploader component.

### /map-manager
This module is responsible for managing (creating, updating, and deleting) maps within the dashboard.

### /map-viewer
DEPRECATED!
This module provides a user interface for viewing and interacting with maps within the dashboard.

### /renderer
Upgraded chunk-based rendering
